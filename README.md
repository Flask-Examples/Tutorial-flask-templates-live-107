## Installation

#### Python Version: 3.8.2

- Check if pyproject.toml file is valid

```sh
poetry check
```

- On base directory of project type:

```sh
poetry env use 3.8
```

- After that

```sh
poetry install
```
---

## How to Run

```sh
poetry run python main.py
```
